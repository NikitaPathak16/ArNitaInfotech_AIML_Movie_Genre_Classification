# ArNitaInfotech_AIML_Movie_Genre_Classification

This repository contains my internship task for ArNita Infotech under the AIML domain.
The goal of this task is to classify the genre of a movie clearly by it's plot.

What This Project Does-
1.Processes the given dataset.
2.Performs the required analysis.
3.Displays results through a simple and user-friendly interface.

Technologies & Tools Used-
1.Python – core programming.
2.Flask – web framework (for UI interaction)
3.HTML & CSS – basic frontend design.

Important Notes
The code is written to be simple and readable.
Proper comments are added for better understanding.
No unnecessary complexity is used.
This project is created only for internship learning and evaluation purposes.
